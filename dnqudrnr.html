GitHub ZIP 자동 반영 도구

ZIP 파일명을 무시하고 내부의 실제 프로젝트 루트를 찾아 GitHub 저장소에 자동 커밋합니다. 단일 HTML 버전이라 GitHub 토큰은 사용자가 직접 입력해야 하며, 기본적으로 토큰은 저장하지 않습니다. 본인 아이폰 전용으로 사용할 때만 “이 아이폰에 저장” 옵션을 켜면 다음 접속부터 자동 입력됩니다.

1. GitHub 연결

단일파일 버전은 GitHub 로그인 버튼 방식이 아닙니다. 안전한 로그인형은 GitHub App + Cloudflare Worker가 필요합니다. 이 파일은 토큰 입력형입니다.
필요 권한: 대상 저장소 Contents Read and write. 기본값은 저장 안 함입니다. 본인 아이폰 전용이면 아래 저장 옵션을 켜서 매번 입력을 줄일 수 있습니다.
저장된 토큰 없음
루트에 반영하려면 / 로 두세요. 특정 폴더에 넣으려면 예: bpw 또는 docs

2. ZIP 선택

index.html이 있는 위치를 우선 프로젝트 루트로 판단합니다. 상위 폴더명은 자동 제거됩니다.

3. 반영 방식

기존 파일 유지 + ZIP 파일만 덮어쓰기
ZIP 안에 있는 파일만 추가/수정합니다. ZIP에 없는 기존 파일은 그대로 남습니다.
선택 위치 전체 교체
선택한 GitHub 위치 안의 기존 파일 중 ZIP에 없는 파일은 삭제됩니다. BPW 버전 교체용으로 편합니다.
ZIP 파일을 선택하면 분석을 시작합니다.
